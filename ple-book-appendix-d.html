Appendix D — Recommended Resources by GPS Category
Foundational texts, organizations, peer networks, and tools that support development in each of the sixteen GPS categories
How to use: Resources are organized by GPS category in tier order. Each category block includes: foundational texts (books and long-form publications that provide conceptual grounding), organizations and peer networks (institutions doing this work that can be learned from directly), and practical tools (frameworks, assessments, and toolkits applicable without significant facilitation expertise). Resources marked with an asterisk (*) are particularly accessible entry points for institutions new to a category. This list is selective rather than exhaustive — it represents the resources most directly relevant to GPS framework implementation, not comprehensive literature reviews.

End of Appendix D — Recommended Resources by GPS Category
Resources marked with an asterisk (*) are recommended entry points for institutions beginning development in a GPS category. They were selected for accessibility, practical applicability, and low cost of access. For each category, start with the asterisked resources before moving to the broader list.

On using these resources in practice: The most effective use of this list is not independent reading but peer learning. The organizations listed under each category are communities of practice — institutions doing this work whose learning is directly applicable and often available through peer relationships rather than only through published materials. Before purchasing books or engaging consultants, identify which peer organizations in your network are strongest in the GPS categories you are developing and request direct peer learning exchanges.
